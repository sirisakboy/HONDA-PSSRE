# HONDA-PSSRE - สมุดภาพอะไหล่ HONDA

แคตตาล็อกอะไหล่ HONDA แบบออฟไลน์ (PSS Retail)

## การใช้งาน

### เปิดดูแบบออฟไลน์
```bash
# เปิดด้วยเบราว์เซอร์
firefox index.html
```

### ดาวน์โหลดเว็บไซต์ใหม่
```bash
./download-website.sh
```

## โครงสร้างโปรเจ็ค

```
HONDA-PSSRE/
├── index.html          # หน้าหลัก
├── css/                # ไฟล์ CSS
├── lib/                # ไลบรารี
├── Images/             # รูปภาพโลโก้และไอคอน
├── HomePageSlide/      # รูปภาพสไลด์หน้าแรก
├── Promotion/          # รูปภาพโปรโมชั่น
└── download-website.sh # สคริปต์ดาวน์โหลด
```

## ทางลัด Desktop

คลิกไอคอน `สมุดภาพ อะไหล่ HONDA` บน Desktop

## หมายเหตุ

- ดาวน์โหลดจาก: https://www.suparat.net/pssretail/
- สำหรับใช้งานแบบออฟไลน์เท่านั้น
- ข้อมูลอาจไม่ทันสมัย โปรดตรวจสอบเว็บไซต์หลักสำหรับข้อมูลล่าสุด
